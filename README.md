# Cyclistic Bike-Share Analysis
**Tools:** Python (Pandas, NumPy, Matplotlib), Tableau

## 📌 Overview
This analysis explores 12 months of trip data to identify behavioral differences between casual riders and annual members. This project is part of the **Google Data Analytics Professional Certificate** capstone.

## 📊 Dashboard Preview
![Tableau Dashboard Preview](/visuals/dashboard.png)

## 📂 Data Source
The raw datasets consist of over 5 million trip records (approx. 1.1 GB). Due to file size limitations, the raw data is not hosted in this repository.
* **Original Source:** [Google Data Analytics Capstone Datasets](https://divvy-tripdata.s3.amazonaws.com/index.html)

## 🚀 How to Use
1. **Data Cleaning:** View `scripts/cyclistic_processing.ipynb` to see how 5M+ rows were handled.
2. **Analysis:** View `scripts/cyclistic_analysis.ipynb` for the statistical breakdown and insights.
3. **Interactive Visuals:** Check the `visuals/` folder for the `.twbx` file or visit my [Tableau Public Profile](https://public.tableau.com/app/profile/jerome.haynes/vizzes).
